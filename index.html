Módulo Para Padres
De Familia

En este módulo podrás realizar el proceso de matrícula y actualizar la información relevante del estudiante de manera sencilla y conveniente. Descubre como este módulo puede facilitar tu experiencia como padre o madre en nuestra comunidad escolar.

Módulo Para Funcionarios

Este módulo está diseñado para funcionarios del colegio, proporcionando herramientas para generar reportes, buscar y recuperar documentos e información, acceder a los registros de los estudiantes, registra y verificar información relevante de manera eficiente. descubre como este módulo puede optimeizar las tareas adnministrativas en nuestra institución educativa.

Módulo Para Profesionales
de la Salud Mental

Este módulo está diseñado para Profesionales de la Salud, ofrece herramientas para generar reportes e informenes estadísticos entre los estudiantes, registrar y actualizar lasintervenciones psicológicas de manera eficaz. descubre cómo este módulo puede mejorar laatención y seguimiento psicológico en nuestra institución educativa.
